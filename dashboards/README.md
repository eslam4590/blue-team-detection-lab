# Security Dashboards

## Purpose
This folder contains visual dashboards designed to help SOC analysts quickly identify suspicious behavior and investigate security incidents.

The focus is on anomalies and patterns rather than raw log volume.

---

## Dashboard Strategy
- Highlight unusual authentication behavior
- Detect spikes and deviations
- Support fast triage and investigation

---

## Included Dashboards

### Anomaly Dashboard
Focuses on:
- Failed login spikes
- High-risk users
- Unusual activity per host

### Login Behavior Dashboard
Focuses on:
- Login activity by time of day
- User authentication trends
- Detection of abnormal login hours

---

## Notes
- Dashboards are built using Kibana visualizations
- Screenshots are provided for documentation purposes
